# Sleep Patterns Dashboard

An interactive Tableau dashboard that analyzes sleep health and lifestyle data to uncover patterns affecting sleep quality. This project uses data visualization techniques to help identify relationships between sleep duration, stress levels, physical activity, heart rate, BMI, and overall health.

## 📌 Project Overview

Sleep is a critical component of overall health and well-being. This project leverages Tableau to transform raw sleep and lifestyle data into meaningful visual insights, enabling users to explore trends and understand factors influencing sleep quality.

The dashboard provides interactive visualizations for analyzing sleep habits, health indicators, and lifestyle factors.



## 📊 Dataset

The dataset includes information such as:

- Age
- Gender
- Occupation
- Sleep Duration
- Sleep Quality
- Physical Activity Level
- Stress Level
- BMI Category
- Heart Rate
- Daily Steps
- Blood Pressure
